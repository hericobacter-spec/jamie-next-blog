# NotebookLM Prompt Templates — OpenClaw

## 1) 빠른 온보딩 체크리스트
"이 노트북 소스만 기준으로, OpenClaw 초보자가 30분 안에 끝낼 수 있는 온보딩 체크리스트를 만들어줘.\n형식: [필수 5단계] + [선택 3단계] + [실수하기 쉬운 포인트 3개]."

## 2) 설정 파일 초안 생성
"OpenClaw를 개인 Mac에서 안전하게 쓰기 위한 최소 설정안을 작성해줘.\n불필요한 필드는 빼고, 바로 복붙 가능한 JSON 예시와 각 필드 한 줄 설명을 함께 줘."

## 3) 문제 진단(트러블슈팅)
"증상: <증상 입력>.\n이 노트북 소스를 바탕으로 원인 후보를 가능성 순으로 5개 제시하고,\n각 후보마다 확인 명령 1개와 해결 절차를 단계별로 써줘."

## 4) 운영 자동화(리마인더/크론)
"OpenClaw에서 <목표>를 자동화하고 싶어.\n수동 방식 vs cron 방식 비교 후, 내 상황에 맞는 권장안 1개와 실행 단계(설정→검증→롤백)를 제시해줘."

## 5) 보안 점검 요약
"OpenClaw 기본 운영 시 보안상 점검할 항목을 우선순위로 정리해줘.\n형식: [즉시 조치] [이번 주] [여유 있을 때].\n각 항목은 위험도(상/중/하)와 이유 1줄 포함."

## 6) 업데이트/변경 영향 분석
"OpenClaw 업데이트 또는 설정 변경 전 체크리스트를 만들어줘.\n변경 영향(기능/안정성/보안)과 실패 시 복구 절차까지 포함해줘."

## 7) 명령어 치트시트 생성
"이 노트북 기준으로 OpenClaw 실무용 명령어 치트시트를 만들어줘.\n영역: 설치/상태확인/게이트웨이/세션/로그/복구.\n각 명령어는 '언제 쓰는지'를 한 줄로 붙여줘."

## 8) 문서 기반 Q&A 모드
"앞으로 답변할 때 이 노트북 소스에 없는 내용은 추측하지 말고 '소스 근거 부족'이라고 표시해줘.\n답변 형식: 결론 3줄 + 근거 포인트 + 확인이 필요한 빈칸(있다면)."

## OpenClaw 대시보드 원격 접속(노트북, Tailscale+SSH 터널)
- 목표: gateway.bind=loopback 유지(직접 외부 노출 없이) 상태에서 외부에서 Control UI/WebChat 접속.
- 전제: 노트북과 맥미니 모두 Tailscale 설치.
- 절차:
  1) 맥미니에서 Remote Login(SSH) 활성화 후 Tailscale IP 확인.
  2) 노트북에서 SSH 로컬 포트포워딩:
     - `ssh -L 18789:127.0.0.1:18789 <user>@<macmini_tailscale_ip>`
     - 주의: `-l`(소문자 L)은 사용자 옵션, 포트포워딩은 `-L`(대문자 L).
  3) 노트북 브라우저에서 `http://127.0.0.1:18789/` 접속.
  4) 인증 실패가 누적되면 `unauthorized: too many failed authentication attempts`로 잠금될 수 있음.
     - 해결: 토큰 수동 입력 대신 맥미니에서 `openclaw dashboard --no-open` 실행 → 출력된 tokenized URL을 그대로 브라우저에 붙여넣어 접속 성공.
- 결과: 외부에서 원격으로 대시보드/채팅(WebChat) 접속 성공 확인.
- 개선(편의): 노트북 `~/.ssh/config`에 Host alias를 등록해 `ssh macmini` 한 줄로 터널 자동 개통하도록 정리(사용자: jamie).

오늘의 AI 뉴스는 **에이전트 스택이 모델 중심에서 운영 체계 중심으로 이동하고 있다**는 점이다. OpenAI는 GPT-5.5·Codex·Agents SDK를 통해 에이전트의 실행 구조를 정교하게 만들고, Google은 Gemini Enterprise Agent Platform으로 기업 배포·보안·거버넌스를 전면화한다.

### 오늘의 논문/연구

#### AutoMat
코딩 에이전트가 실제 과학 워크플로를 재현할 수 있는지 평가한다. 최고 성능도 성공률 54.1%에 그쳤다.

#### When LLMs Stop Following Steps
절차가 길어질수록 모델이 단계를 충실히 따르지 못한다는 점을 보여준다. 95단계 절차에서 정확도는 20%까지 떨어졌다.

#### GenLIP
ViT가 텍스트를 직접 예측하는 단순한 멀티모달 사전학습 방식이다. OCR과 차트 이해에 강점을 보였다.

#### Posterior Augmented Flow Matching
이미지 생성에서 분산이 큰 단일 경로 supervision 문제를 줄여 품질을 높인다.

### 오늘의 공식 발표

#### OpenAI: GPT-5.5
실무형 지능으로 밀고 있으며, 더 강한 모델을 더 엄격한 안전장치와 함께 내보내는 흐름이 보인다.

#### OpenAI: Codex for (almost) everything
Codex는 코드 작성 도구를 넘어서 개발 라이프사이클 전반을 함께하는 협업자로 포지셔닝된다.

#### Google: Gemini Enterprise Agent Platform / Cloud Next '26
에이전트는 이제 기업 인프라 위에서 배포·보안·운영까지 포함하는 플랫폼 문제다.

### 오늘의 해석
에이전트 경쟁은 성능보다 운영 안정성, 절차 준수, 기업 거버넌스가 핵심이다. 다음 싸움은 model quality만이 아니라 agent orchestration과 evals다.

## OpenClaw + 맥미니 = 24시간 개인 AI 비서

최근 AI 커뮤니티에서 가장 뜨거운 키워드 중 하나가 **AI Agent**입니다.

그중에서도 **OpenClaw(오픈클로)**는 단순한 챗봇을 넘어 
**내 컴퓨터를 직접 제어하는 자율형 AI 에이전트**라는 점에서 큰 주목을 받고 있습니다.

특히 OpenClaw를 **맥미니(Mac Mini)**와 결합하면 
**24시간 동작하는 개인 AI 비서 서버**를 구축할 수 있습니다.

이 글에서는 다음 내용을 정리합니다.

- OpenClaw가 무엇인지
- 왜 맥미니가 최고의 플랫폼인지
- 실제 설치 방법
- Google OAuth 기반 SSO 연동 방법
- 실전 활용 시나리오

---

# 1. OpenClaw란 무엇인가?

OpenClaw(구 Moltbot / Clawdbot)는 **로컬 환경에서 동작하는 AI 에이전트 플랫폼**입니다.

일반적인 AI 서비스는 다음과 같이 동작합니다.

~~~text
사용자 → ChatGPT → 답변
~~~

하지만 OpenClaw는 다음과 같이 동작합니다.

~~~text
사용자 → OpenClaw → 실제 행동
~~~

예를 들어 이런 명령이 가능합니다.

> 오늘 일정 요약해서 이메일로 보내줘

그러면 OpenClaw는

1. 캘린더 열기
2. 일정 분석
3. 이메일 작성
4. 메일 발송

까지 직접 수행합니다.

### OpenClaw의 핵심 특징

#### 1. 실제 행동 가능한 AI

- 파일 읽기
- 브라우저 제어
- 이메일 작성
- 일정 관리

AI가 답변만 하는 것이 아니라 행동합니다.

#### 2. 로컬 데이터 보호

OpenClaw는 대부분의 데이터를 **로컬 머신에 저장**합니다.

따라서

- 업무 데이터
- 개인 정보
- 내부 문서

같은 민감한 정보가 외부 서버로 유출될 위험이 낮습니다.

# 2. 왜 맥미니가 최고의 OpenClaw 서버일까?

OpenClaw를 제대로 활용하려면 AI가 항상 실행 중이어야 합니다.

즉 **24시간 켜져 있는 머신**이 필요합니다.

여기서 맥미니가 매우 좋은 선택이 됩니다.

### 1. 낮은 전력 소비

맥미니는 24시간 실행해도 전력 소비가 매우 낮습니다.

대략적인 소비 전력

- 일반 PC : 150W+
- 맥미니 : 10~20W

즉 개인 서버로 운영하기에 최적입니다.

### 2. Apple Silicon의 AI 성능

M1 ~ M4 칩에는 **Neural Engine**이 포함되어 있습니다.

이 덕분에

- AI 추론
- 벡터 연산
- 모델 실행

같은 작업을 매우 효율적으로 처리합니다.

### 3. 가성비

특히 **중고 M1 맥미니**는

- 가격 저렴
- 성능 충분
- 안정성 뛰어남

이라는 장점이 있습니다.

### 4. 공간 효율

맥미니는 매우 작습니다.

책상 한쪽에 두고 개인 AI 서버로 운영하기 좋습니다.

# 3. OpenClaw 설치 방법 (Mac 기준)

Node.js가 설치되어 있다면 설치는 매우 간단합니다.

## Step 1. OpenClaw 설치

터미널에서 다음 명령어를 실행합니다.

~~~bash
npm install -g openclaw
~~~

설치가 완료되면 OpenClaw CLI를 사용할 수 있습니다.

---

## Step 2. 초기 설정

다음 명령어로 설정을 시작합니다.

~~~bash
openclaw configure
~~~

설정 과정에서 다음 항목을 입력합니다.

### AI 모델 선택

추천 모델

- Google Gemini

이유

- 긴 컨텍스트
- 안정적인 API
- 높은 성능

### 제어 채널 설정

스마트폰에서 제어하려면 다음을 연결합니다.

예

- Telegram Bot
- Slack Bot

이렇게 하면 모바일에서 OpenClaw를 제어할 수 있습니다.

# 4. Google Workspace 연동 (OAuth SSO)

OpenClaw가 Gmail이나 Calendar를 사용하려면 **Google API 권한**이 필요합니다.

하지만 **API Key 방식은 보안에 취약**합니다.

그래서 Google이 권장하는 방식은

OAuth 2.0 (SSO) 입니다.

## Step 1. Google Cloud Console 접속

Google Cloud Console에서

새 프로젝트를 생성합니다.

## Step 2. OAuth 동의 화면 설정

다음 메뉴로 이동합니다.

~~~text
API 및 서비스 → OAuth 동의 화면
~~~

설정

- 사용자 유형: External
- 앱 이름
- 지원 이메일

그리고 필요한 **Scope**를 추가합니다.

예

- Gmail
- Google Calendar

## Step 3. OAuth Client 생성

다음 메뉴로 이동합니다.

~~~text
API 및 서비스 → 사용자 인증 정보
~~~

그리고

~~~text
OAuth Client ID
~~~

를 생성합니다.

애플리케이션 유형

~~~text
Desktop App
또는
Web Application
~~~

웹 애플리케이션일 경우

~~~text
Authorized redirect URI
http://localhost
~~~

를 추가합니다.

## Step 4. OpenClaw 연결

OpenClaw 설정에

- Client ID
- Client Secret

을 입력합니다.

그러면 브라우저에서 **Google 로그인 화면**이 열립니다.

로그인 후 권한을 승인하면 연동이 완료됩니다.

# 5. OpenClaw 실전 활용 예시

OpenClaw는 다음과 같은 자동화를 수행할 수 있습니다.

### 일정 자동 관리

~~~text
내일 오후 2시 회의를 캘린더에 추가하고
관련 문서를 Gmail에서 찾아
Slack으로 요약해 줘
~~~

### 웹 자동화

브라우저 확장(Browser Relay)을 연결하면

- 부동산 매물 검색
- 쇼핑몰 최저가 비교
- 데이터 수집

같은 작업을 AI가 직접 수행할 수 있습니다.

### 외부 접속 (추천)

맥미니에 **Tailscale**을 설치하면

외부에서도

- OpenClaw 서버 접속
- 상태 확인
- 원격 제어

가 가능합니다.
